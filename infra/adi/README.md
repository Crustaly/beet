# ADI Public Proof Layer

This layer anchors hashes of encrypted patient reports on ADI testnet.

No raw diagnosis or exact location is anchored publicly.

## Inputs and Outputs

- Input: Encrypted payload generated by hardware device.
- Output: Hash + timestamp + device registration metadata anchored on ADI.

## Commands

- Install ADI CLI (project wrapper):
  - `./scripts/install_adi.sh`
- Anchor a report hash:
  - `./scripts/adi_anchor.sh <sha256-hash> <device-id> <region-general>`

## Required Environment Variables

- `ADI_CLI_BIN` (optional): path to `adi` binary if non-standard.
- `ADI_RPC_URL`: ADI testnet RPC endpoint.
- `ADI_PRIVATE_KEY`: key used by anchoring account.
